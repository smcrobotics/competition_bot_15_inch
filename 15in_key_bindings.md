# Key bindings for SMC 2019-2020 15in robot
## Buttons
### Bumpers
- L1: N/A
- L2: N/A
- R1: LIFT_MOVE_UP
- R2: LIFT_MOVE_DOWN

### Arrows
- up: LIFT_POS_UP
- down: LIFT_POS_DOWN
- left: DRIVE_BRAKE_TOGGLE
- right: TOGGLE_CLAW

### Letters
- X: N/A
- B: LIFT_POS_TOWER_MID
- Y: LIFT_POS_TOWER_LOW
- A: LIFT_POS_TOWER_HIGH
